验证安装环境是否成功

注册github账号

  • 根据注册提示信息进行邮箱验证
  • 进入github进行创建仓库
  • 注意：输入Repository name:yourname.github.io(yourname一定要与你的注册用户名一致,这个就是你博客的域名了)；

    安装hexo

  • hexo是基于node.js的静态博客，官网也是搭建在GitHub上
  • 在电脑上新建一个blog文件夹,该文件夹用于存放你的博客文件,然后右键单击选择“Git Bash Here”
  • 进行安装hexo

安装完成后，再输入命令，验证是否安装正确

初始化hexo

启动本地服务器

打开浏览器，输入localhost:4000,就可以在本地看到你的个人博客了

按Ctrl+C退出服务器

配置config.yml

用编辑器打开config.yml改变deploy的内容；

创建新文章生成一个hello.md文件(/blog/source/_posts/)

用编辑器打开md文件即可编辑文章内容
